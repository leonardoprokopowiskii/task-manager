# 🚀 Task Manager CLI

Gerenciador de tarefas simples e funcional executado via linha de comando, desenvolvido durante a formação **Python** da **Rocketseat**.

## 💻 Sobre o Projeto

Este projeto é uma aplicação de console que permite o gerenciamento de uma lista de afazeres (To-Do List). O objetivo foi aplicar conceitos fundamentais da linguagem Python, como manipulação de coleções, funções e estruturas de repetição.

---

### 🛠 Funcionalidades

- **Adicionar Tarefas:** Registra uma nova tarefa com status pendente.
- **Visualizar Lista:** Exibe todas as tarefas com seus respectivos índices e status visual `[✓]`.
- **Editar Tarefas:** Permite renomear tarefas existentes.
- **Concluir Tarefas:** Marca uma tarefa selecionada como finalizada.
- **Remover Concluídas:** Limpa a lista, deletando todas as tarefas já finalizadas de uma vez.
